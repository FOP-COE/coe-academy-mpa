# COE-241 Handover Guide

## Purpose
This repository is the continuity package for the Power Automate Academy / training workstream.

It keeps the materials needed to understand, replay, and extend the training delivery without carrying the generated clutter that is not useful for future development.

## Project Description

COE-241 is an internal training and enablement initiative built around reusable Power Automate workshop materials, presenter notes, communication assets, and supporting web content.

## What Has Been Kept

- Workshop and presenter guidance.
- Session scripts and recap communications.
- Technical setup and delivery notes.
- Flow export artifacts and Office Script helpers.
- Worklog and evidence of what was delivered.

## What Was Cleaned

- Generated dependency folders.
- Build artefacts and transient compilation files.
- Non-essential runtime outputs.

## Recommended Next Owner Workflow

1. Read the README and the transfer scope first.
2. Review the workshop assets that are still relevant for reuse.
3. Check the flow export and the scripts before making changes.
4. Use the worklog only as historical context, not as the source of truth.
5. Keep only the minimum artifacts needed for the next iteration.

## Continuity Rule

If the project is resumed later, future work should stay focused on reusable training assets and avoid reintroducing generated or duplicated files unless they are required for delivery.
