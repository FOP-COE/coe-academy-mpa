# Transfer Scope

## Included in This Handover
- Workshop delivery materials and presenter assets.
- Session scripts and communication templates.
- Technical setup notes and recovery guidance.
- Flow export artifacts and Office Script helpers.
- Historical worklog evidence and export references.

## Excluded
- Generated dependency folders.
- Build artefacts and transient compilation files.
- Local OS metadata and other non-essential runtime outputs.
- Secrets, credentials, and tenant-specific tokens.

## Validation Recommendation After Clone
1. Confirm the flow package can be opened or imported in the target tenant.
2. Check SharePoint, mailbox, and Power Automate connections.
3. Run one controlled training/session replay.
4. Compare the output against the documented expected behavior.
